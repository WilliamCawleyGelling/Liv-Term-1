Interface for V1 Web Service

Examples of Requests

Get the list of all teams by having:
HTML Method: GET, Resource: teams

Get the teams that have the regular expression:
HTML Method: GET, Resource: teams?tName=<regexp>

Get the list of all players in a teams by having:
HTML Method: GET, Resource: teams/<team>/players

Get information on an idividual player using the pid (Player ID) by having:
HTML Method: GET, Resource: teams/<team>/players/<pid>

Post new information on a player in a team (make sure the team already exists in the database) by having:
HTML Method: POST, Resource: teams/<team>/players
Request Body Example: {"tName" : "Liverpool", "fName" : "Will", "lName":"Cawley Gelling", "nationality" : "English", "DoB" : "2001-04-07" }

Delete information on an idividual player using the pid (Player ID) by having:
HTML Method: DELETE, Resource: teams/<team>/players/<pid>

Edit information on an idividual player using the pid (Player ID) by having:
HTML Method: PATCH, Resource: teams/<team>/players/<pid>
Request Body Example: {"tName" : "Everton", "fName" : "Will", "lName":"Cawley Gelling", "nationality" : "English", "DoB" : "2001-04-07" }
